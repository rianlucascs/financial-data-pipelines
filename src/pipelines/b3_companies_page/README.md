# B3 Companies Page

Pipeline que faz a busca de empresas na página da B3 e coleta as informações exibidas para cada empresa.

Usado para obter dados de empresas que possuem balanço patrimonial registrado na CVM.

Este pipeline **depende dos dados do pipeline `formulario_informacoes_trismestrais`**.

**Site usado:**
[https://sistemaswebb3-listados.b3.com.br/listedCompaniesPage/?language=pt-br](https://sistemaswebb3-listados.b3.com.br/listedCompaniesPage/?language=pt-br)
