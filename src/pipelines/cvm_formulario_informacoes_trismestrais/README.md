
## **Formulário de Informações Trimestrais (ITR)**

Este pipeline baixa e extrai os arquivos públicos do **ITR** disponibilizados pela CVM.

Os arquivos contêm as informações contábeis trimestrais das companhias abertas, incluindo:

* **BPA**
* **BPP**
* **DFC-MD / DFC-MI**
* **DMPL**
* **DRA**
* **DRE**
* **DVA**


O pipeline processa todos os anos disponíveis (2011 até o ano atual), organiza os arquivos e gera a compactação final.
